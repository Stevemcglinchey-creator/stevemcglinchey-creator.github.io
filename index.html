"I built SENwear because I needed it."

My name is Steve McGlinchey. I'm an SNA in a Donegal secondary school. Every day, I saw critical student information get lost in corridor conversations, handwritten notes, and missed messages. So I built a tool that puts everything in one place — so we can focus on what matters: supporting students.

— Steve McGlinchey, SNA & Founder, SENwear
